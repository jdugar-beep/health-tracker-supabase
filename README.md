# Health Tracker Local

A private Vite + React health tracker that saves data locally in your browser using `localStorage`. No Supabase, no login, no backend.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

Upload/import this folder as a normal Vite React app. No environment variables are needed.

## Important

Your data is saved on the specific browser/device where you use the app. It persists after closing the app or reopening it later, but can be lost if you clear browser data, uninstall the PWA, switch devices, or use private browsing. Use the built-in Export Backup button periodically.
